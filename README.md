# The Palace School

This is the official website for The Palace School, built with a modern, full-stack architecture.

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS, GSAP
- **Backend:** Node.js, Express.js, TypeScript
- **Database:** MongoDB, Mongoose

## Project Structure

This project is a monorepo managed with npm workspaces.

- `packages/client`: The React frontend application.
- `packages/server`: The Express backend server.
